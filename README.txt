
# Curriculum Vitae en HTML

Ce projet contient un Curriculum Vitae (CV) réalisé en HTML, offrant une présentation simple et élégante des informations professionnelles et académiques.

## Comment utiliser

1. Clonez ce dépôt sur votre machine locale :
   ```bash
   git clone https://github.com/Paul-FranckDencausse/CV-en-HTML.git
   ```

2. Ouvrez le fichier `index.html` dans votre navigateur web préféré pour visualiser le CV.

## Personnalisation

Vous pouvez personnaliser ce CV en modifiant le contenu du fichier `index.html` selon vos propres informations. Les sections principales comprennent :

- **Informations personnelles** : Nom, prénom, adresse e-mail, numéro de téléphone, etc.
- **Expérience professionnelle** : Détails sur les postes occupés, les entreprises et les périodes d'emploi.
- **Formation académique** : Diplômes obtenus, institutions fréquentées, années d'études.
- **Compétences** : Langages de programmation, logiciels, compétences techniques, etc.
- **Langues** : Maîtrise des langues étrangères.
- **Centres d'intérêt** : Loisirs, activités extra-professionnelles, bénévolat, etc.

## Technologies utilisées

Ce CV a été créé en utilisant uniquement HTML pour assurer une portabilité maximale et une compatibilité avec la plupart des navigateurs web modernes.
